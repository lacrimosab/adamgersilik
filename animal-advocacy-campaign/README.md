# Adamgersilik -- translates from Kazakh as 'humaneness'. This is a website for a
# student project (with 20 members) whose goal is to advocate for stricter regulations  
# against violence toward animals. 

## Files

- `index.html` — one-page landing 
- `style.css` — styles 
- `script.js` — language switcher (KZ/RU/EN) 
- `assets/logo.png` — campaign logo 

## Run locally

Open `index.html` in browser.

## Edit content

- Update page content in `index.html`
- Edit translations in `script.js`
- Adjust colors, spacing, and fonts in `style.css`.
